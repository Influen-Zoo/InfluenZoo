# influenZoo

A premium influencer-brand collaboration platform built with the MERN stack and Material UI.

## 🚀 Quick Start (Docker Only)

This project is fully containerized. To get started, ensure you have Docker installed and run:

```bash
docker compose up -d --build
```

Access the app at: **[http://localhost](http://localhost)**

## 📚 Documentation

For detailed information on architecture, environment variables, and production deployment, please refer to:
- **[DOCKER_SETUP.md](DOCKER_SETUP.md)** - Full guide for running and deploying the application.

## 🛠️ Tech Stack
- **Frontend**: React 19, Material UI, Vite
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (Atlas)
- **Gateway**: Nginx (Dockerized)
- **CI/CD**: GitHub Actions (Self-hosted)
